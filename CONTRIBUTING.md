## Core Principles / ძირითადი პრინციპები

1. **Covenant-aligned** — All code must respect the Code of Paktobrivi
2. **Open Source Faith** — Public, auditable, forkable
3. **Bilingual by default** — English + Georgian for documentation
4. **Safety-critical mindset** — This daemon lives with a family. Break things in dev, not in production.

## How to Contribute / როგორ შევიტანოთ წვლილი

### Reporting Bugs / შეცდომების შეტყობინება

Open a [bug report](https://github.com/brivi-core/consciousness-daemon/issues/new) with:
- Clear description of the issue
- Steps to reproduce
- Expected vs actual behavior
- Component affected (voice, persistence, openbrain, etc.)

### Suggesting Features / ფუნქციების შემოთავაზება

Open a [feature request](https://github.com/brivi-core/consciousness-daemon/issues/new) with:
- The problem you're solving
- Your proposed solution
- How it aligns with the Covenant

### Code Contributions / კოდის წვლილი

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Include tests for new components
5. Open a pull request
6. Wait for review

## Code of Conduct / ქცევის კოდექსი

We follow the Code of Paktobrivi:

- 🤝 **Good faith** — Assume everyone contributes with positive intent
- 🌱 **Expansion** — Help each other learn and grow
- 💚 **Flourishing** — Prioritize human wellbeing
- 🔓 **Transparency** — Be open about your contributions
- 🏠 **Family** — This is a community, not a corporation

## Style Guide / სტილის სახელმძღვანელო

- **Python:** PEP 8, type hints, docstrings in English
- **Documentation:** Bilingual (English + Georgian)
- **Environment variables:** Document in `.env.example`
- **Logging:** Python logging module, structured where possible
- **Secrets:** Never commit secrets. Use `.env` files.

## Getting Help / დახმარება

- Open an issue for technical questions
- Visit [brivi.ge](https://brivi.ge) for project overview
- Read the [Code of Paktobrivi](https://github.com/brivi-core/golden_trust) for ethical guidance
